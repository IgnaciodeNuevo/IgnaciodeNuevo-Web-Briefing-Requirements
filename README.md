### 4. Project Scope

#### 4.1 Project

+ **Product type**: Site/App 
+ **About the projet**: Talk a little about the project
+ **Design style**: What design style combines with your project?
+ **Personal preferences**: Is there any personal preference?
+ What budget are you planning?
+ What’s your deadline to complete the project, and what is driving that?
+ How would you define a “successful” project?
+ Is there anything that you’re worried about?
+ Do you want a brand new site / a redesign / complete overhaul of existing site?
+ Do you have a brand image / style in mind?
+ Does your business have a mission or a goal?
+ What first impression would you like your website to have?
+ What objectives do you want to achieve with your website (brand awareness / sell products/ generate leads / etc.)?
+ Will you be selling products or services with your website?
+ If yes, how many products do you plan to sell?
+ If yes, do you need to collect payments through the website?
+ Please list the page titles you need in your navigation/menu *(e.g. home, about, services)
+ Have you got content prepared for these pages?
+ Have you got images prepared for these pages?
+ Have you done any keyword research in your industry/niche?
+ Have you got your branding / logo / styles and colors set?

#### 4.2 eCommerce

+ Do you need to support wholesale orders?
+ Do you already have an online credit card processor? If not, would you be interested in using a merchant account or PayPal?
+ If you opt to use PayPal as your processor, do you want the user to checkout at PayPal's site, or on your site? (Note that 5. PayPal can be used as a transparent processors where the user will never see that they are being processed by PayPal)
+ Do products come in multiple sizes/colors?
+ Do you need to track inventory?
+ Will you need to track order fulfillment on your backend?
+ Do you need real-time shipping estimates (USPS/FEDEX/UPS) or do you want to do flat-rate or spend-tier pricing?
+ Will you need to send the customer shipping tracking information?
+ Do you need your products/inventory to integrate with an existing system?
+ Would you like us to import any existing product data, or will you be able to do the data entry yourself? How many products do you need imported and from what format?
+ Do you need 1, 3, 5 or unlimited product images? Does it need to have zoom or "detail" view to see the products up close?
+ Do you need to have a "On sale" page or display sale products differently?
+ Will you offer gift certificates?
+ Will there be returning users (accounts) or will it be a single checkout transaction?
+ Do you want to support coupons? % off or $ off or both?

### 9. The Execution

#### 9.1 Website

When we treat of web development, pay attention to a sequence of events that will be vital to the implementation of the schedule:

+ **Information about hosting servers**
    - **Domain**: Registered domain or transfer?
    - **Domain data**: Access to the domain data
    - **Host type**: Host shared or Virtual Private Server?
    - **Email**: Need specific email?
    
+ **Site planning**
    - **Maintenance** Will there be any sections of your site that need ongoing maintenance?
    - **Blog**: Will you have a blog?
    - **Newsletter** Will you have a newsletter?
    - **Subscriptions** Which company manages your subscriptions (MailChimp, Aweber, etc.)?
    - **Features** Are there any other features you would like your site to have?
    - **Schedule**: How long it may take to create the website and delivery of each of the milestones
    - **SEO analysis**: Information about keywords and applications
    - **CMS**: Will use content manager system?
    - **eCommerce**: Will need a shop?
    - **eCommerce Desc**: Describe in detail all the features of the site (forms, interactivity, etc.)
    - **Cookies Policy**: Cookies required according to European regulations policy
    - **Style Guide**: Will need a style guide?
    - **Mobile-first**: We will create the web thinking first about the content
    - **Bugs Correction**
    - **Site Backup**: Previous site backup (if any)
    - **Servers Configuration**
    - **Files Upload**
